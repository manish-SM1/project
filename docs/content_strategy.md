# Content Creation Strategy: "No-Face" Mastery Slides

This document outlines the professional standards for creating slide content for the CareerAI platform.

## 🥇 Best Sources (Safe + Free)
1.  **Official Documentation**: MDN (JS), Python.org (Python), Oracle (Java).
2.  **freeCodeCamp**: For beginner-friendly definitions and rewritten examples.
3.  **MIT OCW / CS50**: For syllabus structure and topic depth.

## 🛠️ The Conversion Method
**Read → Rewrite → Bulletize**
*   **Original**: "Variables are created when you assign a value to it."
*   **Teaching**: "A variable is a name that stores data in memory."
*   **Slide**: 
    1. Stores a value
    2. Helps reuse data
    3. Has a name and a value

## 📋 Standard Slide Template (6–8 Slides)
Every lesson must follow this structure for consistency:
1.  **Title Slide**
2.  **Why it Matters**: Context and real-world use.
3.  **Definition**: Simplified concept.
4.  **Syntax**: Proper code formatting.
5.  **Example**: Practical use case.
6.  **Common Mistakes**: Pitfalls to avoid.
7.  **Practice Question**: Reinforcement.
8.  **Summary**: Key takeaways.

## 🎙️ AI Voice Integration
*   Write speaker notes for every slide.
*   The AI Tutor should guide the transition: *"Now let's understand variables with a simple example..."*

## 💡 Core Philosophy
The platform wins through **Simple Explanations + AI Tutor Guidance + Immediate Practice**.
