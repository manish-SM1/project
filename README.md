CareerAI is an AI-powered career guidance platform that helps students choose the right career path based on their skills, interests, academic performance, and aptitude. 
The system uses Machine Learning and Natural Language Processing (NLP) to provide personalized career recommendations and interactive guidance.

✨ Key Features
🎯 Career Prediction
Predicts suitable career domains using Machine Learning models
Supports algorithms like SVM and Naïve Bayes
📊 Personalized Recommendations
Suggests:
Job roles
Required skills
Certification courses
Higher education paths
🤖 AI Chatbot
NLP-based chatbot for real-time interaction
Answers career-related queries
Provides guidance and suggestions
👤 Student Profiling
Collects and analyzes:
Academic data
Skills
Interests
Aptitude scores
🌐 Web-Based Interface
User-friendly UI
Easy navigation and input
🏗️ System Architecture
User Input → Data Preprocessing → ML Model (SVM / Naïve Bayes)
→ Career Prediction → Recommendation Engine → Chatbot → Output
⚙️ Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Python (Flask / Django)
Machine Learning: Scikit-learn
NLP: NLTK / Transformers
Database: MySQL / SQLite
🧠 Machine Learning Models
Support Vector Machine (SVM) – High accuracy (91%)
Naïve Bayes – Fast and efficient (86% accuracy)
📊 Results
Achieved 91% accuracy using SVM
Provides personalized and relevant career suggestions
Fast response with real-time chatbot interaction
